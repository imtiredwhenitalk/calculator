# Web Calculator

A web-based scientific calculator with Math Lab tools (graphs, matrices, derivatives, and sums).

## Features
- Scientific calculator keypad (trig, logs, powers, constants)
- Function graphing (Chart.js)
- Matrix operations: add, multiply, determinant
- Numerical derivative at x0
- Finite sum over n
- Language toggle (UA/EN)

## Run
Open `main.html` in a browser.

## Math Lab Inputs
- Graph: enter `f(x)` and set min/max range
- Matrix format: `1,2;3,4`
- Sum: expression in `n`, set start/end

## Libraries (CDN)
- Math.js
- Chart.js
- Tailwind CSS
