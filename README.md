# Web Calculator

A web-based scientific calculator with Math Lab tools (graphs, matrices, derivatives, and sums).

## Features
- Scientific calculator keypad (trig, logs, powers, constants)
- Function graphing (Chart.js)
- Matrix operations: add, multiply, determinant
- Numerical derivative at x0
- Finite sum over n
- Language toggle (UA/EN)

## Run
Open `main.html` in a browser.

## Math Lab Inputs
- Graph: enter `f(x)` and set min/max range
- Matrix format: `1,2;3,4`
- Sum: expression in `n`, set start/end

## Libraries (CDN)
- Math.js
- Chart.js
- Tailwind CSS

# Tkinter Calculator

A desktop calculator built with Tkinter. Includes a scientific mode and a function plot panel.

## Features
- Standard operations: add, subtract, multiply, divide
- Scientific functions: sin, cos, tan, log, ln, sqrt, x^2, 1/x, pi, e
- Function plotter with adjustable x-range
- Theme toggle and animated button feedback

## Run
1. Open a terminal in this folder.
2. Run:
   
   ```bash
   python calculator.py
   ```

## Controls
- Keyboard input for digits and operators
- `t` toggles theme
- `s` toggles the scientific panel

## Notes
- The plotter accepts functions like `sin(x)` or `x^2`.
