# Тестування Тригонометричних Графіків

## Як протестувати tan(x) та cot(x)

### Відкрийте калькулятор
1. Відкрийте `main.html` у браузері
2. Перейдіть на вкладку **"Матем. лабораторія"**
3. Виберіть вкладку **"Графік"**

---

## Тест 1: Тангенс (tan)

### Параметри:
- **Функція:** `tan(x)`
- **X від:** `-3.14`
- **X до:** `3.14`

### Очікуваний результат:
- Періодичний графік з вертикальними розривами
- Асимптоти при x ≈ ±1.57 (±π/2)
- Графік проходить через (0, 0)
- Значення зростають від -∞ до +∞ між асимптотами

---

## Тест 2: Котангенс (cot)

### Параметри:
- **Функція:** `cot(x)`
- **X від:** `0.1`
- **X до:** `6.28`

### Очікуваний результат:
- Вертикальні розриви при x ≈ 3.14, 6.28 (π, 2π)
- Графік спадає від +∞ до -∞ між асимптотами
- Періодичність π (на відміну від 2π у sin/cos)

---

## Тест 3: Комбінована функція

### Параметри:
- **Функція:** `sin(x) + tan(x)`
- **X від:** `-3`
- **X до:** `3`

### Очікуваний результат:
- Комбінований графік з розривами від tan(x)
- Хвиляста крива з асимптотами

---

## Тест 4: Калькулятор (прямі обчислення)

### На панелі Калькулятора:
1. Натисніть `tan` → `(` → `0` → `)` → `=`
   - **Очікується:** `0`

2. Натисніть `AC` → `ctg` → `(` → `pi` → `/` → `4` → `)` → `=`
   - **Очікується:** `1` (cot(π/4) = 1)

3. Натисніть `AC` → `tan` → `(` → `pi` → `/` → `4` → `)` → `=`
   - **Очікується:** `1` (tan(π/4) = 1)

## Технічні деталі

### Реалізація:
- **Функція cot:** `cot(x) = 1/tan(x)`
- **Кількість точок:** 480 (для плавності)
- **Фільтр:** Значення > 10⁶ обрізаються для запобігання артефактів
- **Розриви:** `spanGaps: false` у Chart.js

### Файли:
- `script.js` - логіка обчислення та графіки
- `main.html` - кнопка `ctg` у калькуляторі
- `README.md` - повна документація

---

## Можливі проблеми

### Якщо графік не відображається:
1. Перевірте консоль браузера (F12)
2. Переконайтесь, що CDN бібліотеки завантажились (Math.js, Chart.js)
3. Спробуйте оновити сторінку (Ctrl+F5)

### Якщо розриви не видно:
- Це нормально - асимптоти можуть бути за межами діапазону
- Спробуйте змінити діапазон X, щоб включити π/2 (1.57) для tan
- Або π (3.14) для cot

---

## Приклади для копіювання

**Готові функції для тестування:**
```
tan(x)
cot(x)
tan(x) + cot(x)
1/tan(x)
sin(x)/cos(x)
```

**Діапазони:**
- **tan:** від `-3.14` до `3.14`
- **cot:** від `0.1` до `6.28`
- **обидва:** від `-6.28` до `6.28`
